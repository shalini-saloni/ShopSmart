# ShopSmart — Clothing Website 

## 1) Overview
**ShopSmart** is a modern clothing e-commerce website where users can browse and shop fashion items such as **Men’s Wear, Women’s Wear, Kids Wear, Winter Collection, Accessories**, and **New Arrivals**.

The system includes a responsive storefront and an admin panel where products can be managed using a full CRUD REST API.

---

## 2) Problem Statement
Online clothing shopping becomes difficult because:
- too many choices without clean filtering
- unclear product information like sizes and materials
- slow browsing and poor UI on mobile
- stock and pricing may not be updated

Store owners also need:
- a simple dashboard to add/update/remove products
- a reliable database with easy management

---

## 3) Solution
ShopSmart solves this by providing:
- clean UI with category-based shopping
- filters for size, price, category, and color
- product details with images, size options, material info
- admin CRUD dashboard to manage inventory easily
- scalable backend with Prisma ORM and SQLite database

---

## 4) Target Users
### Customers
- people looking for trendy fashion
- students and daily wear shoppers
- buyers searching outfits for occasions

### Admin / Store Owner
- manages product listings, pricing, stock, images

---

## 5) Core Features

### Customer Side (MVP)
- Home page with featured categories + new arrivals
- Products page with search + filters
- Product detail page
- Add to cart (basic)
- Checkout page (basic without payment)

### Admin Side (MVP)
- Dashboard
- **Product Management (Full CRUD)**
  - Create Product
  - Read Products
  - Update Product
  - Delete Product

---

## 6) Tech Stack
### Frontend (Vercel)
- Next.js + TypeScript
- TailwindCSS
- Fetch/Axios API calls

### Backend (Render)
- Node.js + Express
- Prisma ORM
- SQLite3 Database
